This document specifies the architecture of Handle-based FAIR Digital Objects (FDOs). FDOs are persistent bundles of information identified by globally unique and resolvable persistent identifiers (PIDs) that resolve to structured FDO Records defined by explicit profiles and registered attributes. This document specifies how the core building blocks of FDOs, as described in the technology-agnostic FDO Specifications, are realized by Handle-based FDOs.

This document is an official publication of the FDO Forum. This document version has the status of a draft work document according to the FDO Forum process. This document will undergo the FDO Forum process for official documents.
