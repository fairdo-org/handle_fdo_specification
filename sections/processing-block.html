Implementation Detection
An string is detected to identify a Handle-based FDO if and only if
  1. the string is resolved into an FDO record that is valid at least against the 0.FDO/Root profile (i.e., validation as described in [H6-2] is successful against the 0.FDO/Root profile).
Validation
Let `s` be a string. The following steps need to be taken to determine that `s` references a valid, Handle-based FDO.
  1. Detect `s` to be a handle identifier according to [H1-2].
  2. Resolve `s` into a Handle record `h`.
  3. Look for all attributes `a1,...,an` in the FDO record with `ai,def=0.FDO/Profile`. For each of those attributes,
