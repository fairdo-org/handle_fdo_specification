Each section within this chapter explains in detail how the FDO building blocks from the technology-agnostic FDO specification are implemented by Handle-based FDOs.

PID

[H1-1] PID System
Each Handle-based FDO has a Handle identifier as its PID, assigned using the Handle System.
[H1-2] PID Resolution
The resolution of a Handle-based FDO, which is identified by PID X, into an FDO record uses the global Handle Resolver to receive the JSON record at `https://hdl.handle.net/api/handles/[X]`.
