[S-1] PID Detection

Input:

String `s`.

Process:

Check whether `s` matches the pattern from [H1-2]. On success, return outputs below. If it does not match, this cannot be a Handle-based FDO - Stop. 
function detectPID (input: string) -> PIDCandidate | undefined {
    const REGEX = new RegExp('^([0-9A-Za-z])+.([0-9A-Za-z])+/([!-~])+$');  // regular expression from [H1-2]

    if (REGEX.test(input)) { // Check if text matches
        return new PIDCandidate(input, 'https://hdl.handle.net/api/handles');
    } else {
        return undefined; // This cannot be a Handle-based FDO. Stop!
    }
}

Output:

On success:
  1. the PID candidate `s`
  2. the global handle resolver `hdl.handle.net/api/handles`
