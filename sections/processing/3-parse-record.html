[S-3] PID Record Parsing into an FDO Record

Input: JSON

Process: Map all handle values from the JSON to FDO attributes as described in [H2-2].

Output: List of attributes a1=(a1,def,a1,val ),..., an=(an,def,an,val ). The ai,def can either be attribute definition PIDs or attribute names.
