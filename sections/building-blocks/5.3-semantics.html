[H5-3] Semantics
A name must be and a description may be included in a Handle-based attribute definition using the attributes 21.T11969/Name respectively 21.T11969/Description in the FDO record. If a JSON schema is used to encode syntax/union/composition (see [H5-4] - [H5-6]), the name and the description can be repeated in the JSON schema using the keywords title and description.
