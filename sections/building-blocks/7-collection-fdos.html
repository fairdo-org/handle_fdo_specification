Collection FDOs

Collection FDOs refer to one or more other FDOs. This is useful to organize and connect FDOs in a hierarchical manner (e.g., when describing a dataset and all files included within it). A collection FDO MUST comply to the 0.FDO/Root profile like any other FDO. It also has a 0.FDO/Data attribute that uses attribute referencing to refer to the FDOs that are included in the collection. This enables semantically correctly described relationships to other FDOs using the normal FDO referencing mechanisms.

Lets assume one FDO that describes a dataset (aka. dataset FDO) and five FDOs for single files (aka. file FDO). Every file FDO uses the root profile and adds some more information about the file it describes (e.g., file name, file size, file format), but also the HTTP URL to access it. The file FDO records could look similar to this:

PID of Attribute Definition Example Value
0.FDO/Profile 0.FDO/Root
0.FDO/Type 21.example/fileFormat
0.FDO/Data 21.example/HTTPURL
21.example/fileFormat image/jpeg
21.example/HTTPURL https://test.example/image01.jpg
21.example/fileSizeInKB 4242
0.FDO/Name Placeholder01
21.example/isPartOf 21.example/dataset
File FDO record (fictionary PID: 21.example/file01)

The dataset FDO with the PID '21.example/dataset' is a collection FDO referring to all file FDOs. Furthermore, it has some additional information about the dataset (e.g., dataset title, creation date). The 0.FDO/Type attribute refers to the profile used in the FDO, but could also refer to any other attribute or an FDO defining an FDO type. The dataset FDO record could look similar to this:

PID of Attribute Definition Example Value
0.FDO/Profile 0.FDO/Root
0.FDO/Type 0.FDO/Profile
0.FDO/Data 21.example/hasPart
21.example/hasPart 21.example/file01
21.example/hasPart 21.example/file02
21.example/hasPart 21.example/file03
21.example/hasPart 21.example/file04
21.example/hasPart 21.example/file05
0.FDO/Name Example dataset
21.example/creationDate 2026-06-01T12:00:00Z
Dataset Collection FDO record (fictionary PID: 21.example/dataset)
