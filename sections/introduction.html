Introduction

The Handle system (see also [[RFC3650]], [[RFC3651]], and [[RFC3652]]) is a distributed system for globally unique, persistent, and resolvable identifiers (PIDs). It manages PIDs and provides exactly one corresponding [=PID Record=] to every PID. This PID record is an indexed list of key-value pairs in which information can be stored. The Handle system provides an access control mechanism for every [=PID record=]. Many existing systems in climate research and the DOI system [[ISO26324]] use the Handle system.

The Handle-based FDO Specification details

  1. how the building blocks from the FDO Specification are realized with Handle-based FDOs, and
  2. how a machine processes a Handle-based FDO.
