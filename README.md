# Handle Implementation Specification of the [FDO Specifications](https://fairdo-org.github.io/fdo-architecture-spec/)

This repository contains the **Handle Implementation Specification**, developed and maintained by the **TSIG Working Group** of the **FDO Forum**.

The specification describes how the [FDO Specifications](https://fairdo-org.github.io/fdo-architecture-spec/) are implemented in practice using the [Handle System](https://www.handle.net/). 
It defines the mapping of generic FDO concepts to Handle components and adapts the building blocks and processing steps from the FDO specifications. 
The specification serves as a technical reference for implementers, reviewers, and contributors working on FDO implementations based on the Handle System.

The authoritative rendered version of the specification is published via [**GitHub Pages**](https://fairdo-org.github.io/handle_fdo_specification/).
